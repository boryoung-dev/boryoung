## 구현 중인 개선사항

### 1. 상품 소개 탭 리치 에디터 (완료)
- ContentTab의 content textarea → TiptapEditor로 교체
- 이미지 업로드 + 글 교차 입력 가능
- 기존 excerpt (요약), 포함/불포함 리스트는 유지

### 2. 일정 여러 이미지 갤러리 (완료)
- DB: Itinerary 모델에 `imageUrls Json @default("[]")` 추가
- Admin: 여러 이미지 업로드 UI (가로 썸네일 + 추가/삭제)
- Web: 가로 스크롤 갤러리 (레퍼런스: 닌빈 여행 사진 스타일)
- 기존 imageUrl 하위 호환 유지

### 3. SEO 자동 생성 (예정)
- 상품 등록 시 title+destination+holes 기반 Meta Title/Description 자동 채움
- 수동 수정 가능하되, 빈 값이면 자동 생성

### 4. 태그 자동 매핑 (예정)
- 카테고리+난이도+홀수 기반 태그 자동 선택
- 운영자가 추가/제거 가능

### 5. 등록 플로우 간소화 (예정)
- 9탭 → 5탭 축소
- 설정 탭 → 기본 정보 하단 토글로 통합
- 태그/SEO 탭 → 자동화 후 고급 설정으로 이동

## 운영 등록 플로우 개선 제안

현재 탭: 기본 정보 → 상품 소개 → 이미지 → 일정 → 가격 옵션 → 출발일정 → 태그/SEO → 설정 → 페이지 편집

### 필수 (운영자가 직접 입력해야 하는 것)
| 탭 | 이유 |
|----|------|
| 기본 정보 | 상품명, 카테고리, 목적지, 가격, 인원, 항공사 등 비즈니스 핵심 데이터 |
| 이미지 | 실제 골프장/숙소 사진은 사람이 선택해야 함 |
| 일정 | 일자별 상세 일정은 상품마다 다르고 정확해야 함 |
| 가격 옵션 | 옵션별 가격 책정은 비즈니스 결정 |
| 출발일정 | 실제 가용 날짜는 항공/골프장 예약 상황에 따라 다름 |

### 자동화 가능 (시스템이 처리)
| 탭 | 현재 | 개선안 |
|----|------|--------|
| **태그/SEO** | 수동 입력 (자동생성 버튼 있음) | 등록 시 자동 생성. 제목+목적지+홀수로 Meta Title/Description 자동 채움. 태그도 카테고리+난이도+홀수 기반 자동 매핑 |
| **설정** | isActive/isFeatured/sortOrder 수동 | 기본값(active=true, featured=false, sortOrder=0)으로 충분. 탭 제거하고 기본 정보 하단에 토글 2개만 |
| **Slug** | 자동생성 (readonly) | ✅ 이미 자동화됨 |
| **상품 소개 - 포함/불포함** | 매번 수동 입력 | 카테고리별 템플릿 제공. "발리" 카테고리 선택 → 자주 쓰는 포함/불포함 자동 채움 |

### 나중에 추가 (등록 후 편집)
| 탭 | 이유 |
|----|------|
| **페이지 편집** | 상세 HTML 콘텐츠는 기본 등록과 별개. 등록 후 편집 화면에서 천천히 꾸미면 됨. 초기 등록 시 일정+이미지만 있으면 상품 페이지 렌더링 가능 |
| **상품소개 - 이미지+글 교차** | contentHtml에서 이미 img 태그 지원. TiptapEditor에 이미지 삽입 기능 추가하면 됨. 등록 후 편집에서 보강 |

### 제안: 간소화된 등록 플로우 (5단계)
```
1. 기본 정보 (제목, 카테고리, 가격, 인원 등)
   └─ 설정 토글 2개(공개/추천) 하단에 포함
   └─ SEO 자동생성 (백그라운드)
   └─ 태그 자동매핑 (카테고리 기반)

2. 상품 소개 (요약, 포함/불포함)
   └─ 카테고리별 템플릿 자동 채움

3. 이미지 (드래그&드롭 다중 업로드)

4. 일정 (일자별 제목/설명/식사/숙소/골프/이동)

5. 가격 옵션 + 출발일정 (한 탭에 합치기)
   └─ [등록] 클릭
```

→ 9탭 → 5탭으로 축소
→ 등록 후 `/products/{id}/edit`에서 페이지 편집, SEO 미세조정, 태그 변경 가능

### 추가 개선 아이디어
- **엑셀 일괄 등록**: 여행사에서 엑셀로 상품 관리하는 경우가 많음. 엑셀 업로드 → 다수 상품 일괄 등록
- **상품 복제**: 기존 상품을 복제해서 날짜/가격만 변경 (가장 빠른 등록 방법)
- **카테고리별 일정 템플릿**: "발리 4박6일" 선택하면 기본 일정 자동 채움
- **이미지 라이브러리**: 한번 올린 골프장 사진을 재사용 가능하도록
